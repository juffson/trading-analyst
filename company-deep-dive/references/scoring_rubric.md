# 11 维度打分标准（1-5 分）

每个维度 1-5 分，总分 55 分。打分必须**附证据**，不能凭感觉。

---

## 1. 业务简单性

| 分数 | 标准 | 示例 |
|------|------|------|
| 5 | 小学生都能听懂，一句话讲完 | 可口可乐（卖甜水）、茅台（卖白酒）、分众（电梯广告） |
| 4 | 高中生能听懂 | 顺丰（快递）、美的（家电） |
| 3 | 大学生需要解释 | 腾讯（社交+游戏+支付+投资） |
| 2 | 需要专业知识 | 一家综合性投行 |
| 1 | 讲不清楚 | 多元化控股集团（软银、伯克希尔—但伯克希尔有例外） |

**关键问题**：你能在 2 分钟内给妈妈讲清楚这家公司靠什么赚钱吗？

---

## 2. 经济模型

综合 economic_model_analyst 的 5 个子项评分（盈利/成长/营运/偿债/现金流质量）。

| 分数 | 标准 |
|------|------|
| 5 | ROE 长期 >20%，毛利率 >50%，FCF / 净利润 >100%，偿债压力极低 |
| 4 | ROE 15-20%，毛利率 30-50%，FCF 质量健康，偿债能力强 |
| 3 | ROE 10-15%，行业平均水平，偿债无明显风险 |
| 2 | ROE 5-10%，毛利率薄，或现金流不佳 |
| 1 | 亏损或 ROE <5%，现金流持续为负，高负债 |

加分项：5 年趋势 ↗
减分项：5 年趋势 ↘ 或指标波动巨大

---

## 3. 商业模式

| 分数 | 标准 |
|------|------|
| 5 | 定价权强、经常性收入、客户留存率高、资本效率高（轻资产） |
| 4 | 明确的盈利逻辑 + 合理的定价权 |
| 3 | 中规中矩，典型行业利润率 |
| 2 | 重资产 + 激烈价格竞争 |
| 1 | 商业模式存疑（烧钱换规模且看不到盈利路径） |

**关键问题**：
- 公司靠什么赚钱？卖产品？卖服务？卖广告？卖数据？
- 为什么客户愿意付钱？为什么付这个价？
- 客户 10 年后还会付钱吗？

---

## 4. 企业文化与基因

| 分数 | 标准 |
|------|------|
| 5 | 清晰的使命愿景 + 长期贯彻 + 曾经帮公司渡过重大危机 + 创始人精神延续 |
| 4 | 有明确使命愿景，管理层长期坚持 |
| 3 | 有使命愿景但不够特别，或实际行动与口号有落差 |
| 2 | 企业文化模糊，频繁战略摇摆 |
| 1 | 高管频繁变动，文化混乱，财务造假/管理层丑闻历史 |

**关键问题**：
- 创始人是什么出身？公司是做什么起家的？
- 公司经历过什么危机？怎么渡过的？
- CEO 公开讲话和股东信里反复强调什么？做到了吗？
- 公司的核心价值观在日常运营中能看到吗？

---

## 5. 护城河

五大护城河类型（巴菲特/Pat Dorsey 分类）：
1. 无形资产（品牌、专利、牌照）
2. 客户转换成本
3. 网络效应
4. 成本优势（规模经济、地理优势、工艺）
5. 有效规模（自然垄断）

| 分数 | 标准 |
|------|------|
| 5 | 2 种以上强护城河 + 10-20 年内难被颠覆 + 趋势 ↗ |
| 4 | 1 种强护城河 + 趋势稳定或增强 |
| 3 | 有一些竞争优势但脆弱，或趋势不明 |
| 2 | 护城河正在被侵蚀（新技术/监管/替代品） |
| 1 | 无明显护城河或完全同质化 |

**关键问题**：
- 如果竞争对手拿到 100 亿资金，能在 3 年内复制公司的优势吗？
- 10 年后这家公司还在吗？还是这个样子吗？
- 它的未来是否依赖某个不可预测的技术突破或政策变动？（如果是，扣分）

---

## 6. 管理团队

| 分数 | 标准 |
|------|------|
| 5 | 诚实正直 + 战略远见 + 卓越执行力 + 长期持股 + 优秀资本配置记录 |
| 4 | 具备其中 4 项 |
| 3 | 具备其中 2-3 项，或管理层相对平庸但没大问题 |
| 2 | 有 1-2 个重大污点（资本配置差/对赌输了/关联交易多） |
| 1 | 造假 / 频繁诉讼 / 激励机制严重扭曲 |

**验证方式**：
- 读过去 5 年股东信/业绩会实录
- 查管理层持股比例和变动
- 看历史大规模并购的 ROI
- 查有没有未披露重大诉讼

---

## 7. PESTEL

综合 pestel_analyst 的 overall_score。动态趋势权重：
- 如果 6 个维度全部 ↗ 或稳定，至少 4 分
- 如果 2 个以上 ↘ 且驱动力短期无法逆转，最多 3 分

| 分数 | 标准 |
|------|------|
| 5 | 6 个维度中至少 4 个利好 + 无重大利空 |
| 4 | 大部分利好，个别中性 |
| 3 | 利好利空相当 |
| 2 | 2-3 个维度明显逆风 |
| 1 | 宏观环境对公司严重不利（如监管收紧 + 需求萎缩 + 技术被颠覆） |

---

## 8. 动态波特五力

综合 porter_five_forces_analyst 的 overall_score。

| 分数 | 标准 |
|------|------|
| 5 | 6 力中至少 4 个处于强势位置 + moat_durability_years ≥ 15 |
| 4 | 多数力度有利 + moat_durability_years ≥ 10 |
| 3 | 行业格局中性 + moat_durability_years 5-10 |
| 2 | 多数力度不利或趋势 ↘ |
| 1 | 行业格局严重恶化（如被技术颠覆、低价搅局者涌入） |

---

## 9. 安全边际

基于主代理算出的公司总价值（DCF 中性 + 账面资产）对比当前股价。

| 分数 | 标准 |
|------|------|
| 5 | 股价 < 中性价 × 50%（半价以下，深度低估） |
| 4 | 股价在中性价 50-70% |
| 3 | 股价在中性价 70-90% |
| 2 | 股价在中性价 90-110%（合理区间） |
| 1 | 股价 > 中性价 × 110%（高估） |

**特别注意**：
- 高质量公司（前 8 项合计 ≥ 35 分）可以接受 70-90% 的溢价
- 低质量公司（前 8 项合计 < 30 分）即便深度低估也要谨慎
- 悲观情景估值应作为底线参考

---

## 10. 第二层思维

一层思维：公司被低估，应该买。
二层思维：为什么市场不买？市场是错了还是我错了？如果市场错了，什么时候会纠正？

| 分数 | 标准 |
|------|------|
| 5 | 发现市场误解的结构性原因 + 有明确 3-6 个月内可验证的催化剂 |
| 4 | 识别了市场低估的暂时性原因（如行业周期底部 + 数据指标将在下季度改善） |
| 3 | 能说出一些低估原因但逻辑不够扎实 |
| 2 | 找不到明显的低估理由 or 理由牵强 |
| 1 | 市场正确，公司真的没戏（这是"价值陷阱"） |

**关键问题**：
- 为什么市场给它这么低的估值？谁卖的，为什么卖？
- 这些理由是暂时的（疫情、行业周期、管理层换届、短期利空新闻）还是结构性的（商业模式被颠覆）？
- 如果是暂时的，要监控什么指标来确认问题正在解决？

---

## 11. 致命风险

| 分数 | 标准 |
|------|------|
| 5 | 最坏情况下公司仍然能生存（账面现金充足、无刚性兑付、多元化业务） |
| 4 | 致命风险发生概率 <5%，且管理层有明确应对方案 |
| 3 | 存在一些重大风险但有缓解措施 |
| 2 | 存在高概率的重大风险（如单一大客户依赖 + 正在流失） |
| 1 | 黑天鹅来了就完蛋（单一产品、单一市场、高杠杆、ESG 踩雷隐患） |

**必须列出 Top 3 致命风险**，每个都要说：
- 风险是什么？
- 发生的概率（你的主观估计）
- 发生后公司能否活下来？损失多少价值？
- 管理层在年报/业绩会上如何提及这个风险？如何应对？

## 综合评分与投资建议

### 分数加权（可选）

所有 11 项等权（每项 5 分，总分 55），或可自定义权重。默认推荐：
- 护城河、经济模型、管理团队、安全边际 权重 1.5x（更重要）
- 其他项权重 1.0x

简单版：直接加总。

### 建议档位

| 总分 | 建议 |
|------|------|
| ≥ 45 | 强烈推荐深度研究，可考虑建仓 |
| 40-44 | 优质公司，等更好买点（股价到中性 × 70% 以下） |
| 35-39 | 中性，建议持续跟踪 |
| 30-34 | 暂不考虑 |
| < 30 | 基本面/估值有重大问题，避免 |

**必须说明的事**：
1. 建议买入价格区间（基于中性估值的 60-70%）
2. 核心跟踪指标（未来 4 个季度看哪些数据）
3. 退出信号（出现什么情况应卖出）
4. 风险提示

## 打分防偏技巧

1. **别被名气带偏**：茅台是茅台，但不代表它当下值得买。评分要基于当前情况和未来 10 年预期，不是历史光环。
2. **别被你的持仓带偏**：如果你持有这只股票，打分更严格一点；如果不持有，更中立。
3. **用红队思维**：每个 4-5 分的维度都问"有没有可能我错了？"
4. **用灰度思维**：打 3 分不丢人，模糊的正确好过精确的错误。
5. **写下理由**：每个打分必须 1-3 句话说明理由，逼自己思考。

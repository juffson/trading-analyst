# HTML 决策仪表盘结构规范

所有深度价值分析报告都要渲染为单一 HTML 文件（内联 CSS/JS/Chart.js），放到用户的 workspace folder 里。

## 12 个必须区块（按顺序）

### 1. 报告标题区
- 公司中英文名 + 交易所 + 代码
- 行业分类
- 报告生成日期
- 右上角：当前股价 + 当日涨跌幅

### 2. 综合评分横幅
- 11 维度总分（例如 42/55）
- 推荐档位色块：
  - 强烈推荐：绿色 `#16A34A`
  - 优质公司等买点：浅绿 `#84CC16`
  - 中性跟踪：黄色 `#EAB308`
  - 暂不考虑：橙色 `#F97316`
  - 避免：红色 `#DC2626`
- 一句话结论

### 3. 11 维度雷达图 + 评分卡
- 左侧：Chart.js radar chart（1-5 刻度）
- 右侧：11 个维度分数卡（每个卡片包含分数 + 一句话理由）

### 4. 财务五年趋势图（2×3 网格）
每个子图是 Chart.js 折线图，X 轴年份，Y 轴数值：
- ROE (%)
- 营收（亿）+ 同比增速（双轴）
- 净利润（亿）+ 同比增速（双轴）
- 毛利率 / 净利率（同图两条线）
- 自由现金流（亿）
- 资产负债率（%）

### 5. 三档 DCF 估值对比
- 横向条形图（水平）：
  - 乐观（Bull）每股价值
  - 中性（Base）每股价值
  - 悲观（Bear）每股价值
  - 当前股价（虚线或特殊标记）
- 附账面资产价值（每股）
- 附公司总价值表格

### 6. 安全边际可视化
- 价格尺：把当前股价放在 [悲观价, 乐观价] 刻度上
- 安全边际百分比显示
- 色带：红（<当前价）/ 绿（>当前价）

### 7. 护城河五力象限图
- 5 个护城河类型（品牌/转换成本/网络效应/成本优势/规模）
- 每个类型：强度条（1-5）+ 趋势箭头

### 8. PESTEL 六维动态分析
- 六边形雷达图（当前分数）
- 每个维度下方：趋势箭头（↗/↘/→）+ 关键驱动力 tag

### 9. 波特六力蜘蛛图
- 六力雷达（5 力 + 第六力互补品）
- 每个力下方：趋势箭头 + 关键观察

### 10. 第二层思维卡片
- 市场观点 vs 分析师观点（并列对比）
- 催化剂时间线
- 需要验证的指标清单

### 11. 致命风险清单
- Top 3 风险（每个：风险描述 + 概率 + 损失评估 + 管理层应对）
- 可折叠的完整风险矩阵

### 12. 结论与行动计划
- 买入价格带（基于中性估值 × 60-70%）
- 建议仓位
- 跟踪指标（未来 4 个季度）
- 退出触发条件
- 数据来源 + 时间戳 + 免责声明

## 设计规范

### 颜色
- 主色：`#1E40AF`（深蓝，专业感）
- 成功：`#16A34A`（绿，利好/买入）
- 警告：`#EAB308`（黄，中性）
- 危险：`#DC2626`（红，利空/卖出）
- 背景：`#F9FAFB`
- 卡片：`#FFFFFF` + `border: 1px solid #E5E7EB`
- 文字：主 `#111827`，辅 `#6B7280`

### 字体
- 标题：`font-weight: 600`，14-24px
- 正文：`font-weight: 400`，13-14px
- 数值：`font-weight: 500`，等宽字体 `monospace`
- 中文：`"PingFang SC", "Helvetica Neue", Arial`

### 布局
- 桌面：容器 1200px，12 栅格
- 移动：单列堆叠
- 区块间距：24px
- 卡片内边距：16px

### 图表
- 用 Chart.js 4.x（从 CDN 引入）
- 颜色沿用主色系
- 所有图表都要有图例 + tooltip
- 单位在图例中明确（亿元 / % / x）

### 可交互
- 所有卡片可 hover 微动效
- 长文本可折叠（"展开更多"）
- 顶部固定导航（点击跳转区块）

## 文件命名

格式：`<公司名>_价值分析_<YYYY-MM-DD>.html`

示例：
- `贵州茅台_价值分析_2026-04-21.html`
- `腾讯控股_价值分析_2026-04-21.html`
- `Apple_Value_Analysis_2026-04-21.html`

保存位置：`/Users/sirius/Desktop/daily-work/`（或用户指定的 workspace）

报告完成后用 `present_files` 工具向用户展示，并用 `computer://` 链接让用户直接打开。
